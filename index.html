Rekkid Seller Tools

Rekkid Seller Tools is a private application used to manage and optimise the Rekkid Etsy store.

The tool integrates with the Etsy API to access shop data including listings, orders, and sales performance. It is used to analyse store performance, maintain listing consistency, and streamline updates across products.

This application is not publicly available and is used solely for internal store management purposes.

Data accessed through the Etsy API is only used to support store operations and is not shared with third parties.

Privacy Policy

This application accesses Etsy shop data including listings, orders, and shop performance metrics for the purpose of managing and improving store operations.

No personal customer data is stored outside of Etsy systems. Any data accessed is used strictly for internal analysis and operational improvements.

No data is sold, shared, or distributed to third parties.

If you have any questions about data usage, please contact hello@rekkid.store.
