Rekkid Seller Tools

Private internal tooling used to manage, monitor, and optimise the Rekkid Etsy store operations. This page provides transparency about how the application works and what data it accesses.

⚙

What is Rekkid Seller Tools?

Application overview

Rekkid Seller Tools is a private, internal application built to help the Rekkid Etsy store run more efficiently. It integrates with the Etsy API to give us a centralised view of our shop's performance — something the default Etsy seller dashboard doesn't fully provide.

The application is used exclusively by the Rekkid store owner for day-to-day store management. It is not a public product, not available to other sellers, and has no external users.

📈

Why do we need API access?

Business justification

Running a growing Etsy store requires staying on top of inventory, pricing, listing performance, and order fulfilment. The native Etsy tools are limited and don't support the kind of analysis and automation that helps us provide a great customer experience.

API access allows us to:

  • Monitor listing performance and identify which products need attention
  • Track inventory levels across our product variants in real time
  • Review order status and fulfilment metrics to ensure timely dispatch
  • Analyse sales trends to make informed decisions about new products
  • Maintain listing consistency — descriptions, tags, pricing — across our catalogue
🔒

API scopes requested

Data access permissions

The application requests only the scopes necessary to support store management functions. No scope is requested without a clear operational purpose.

Scope Purpose
listings_r Read listing details, pricing, descriptions, and active status to monitor and audit our catalogue
listings_w Update listing details such as descriptions, tags, and pricing for bulk maintenance tasks
transactions_r Read order and transaction data to track sales performance and fulfilment
shops_r Read shop-level data including stats and settings for performance reporting
👤

Privacy Policy

How we handle data

Rekkid Seller Tools accesses Etsy shop data solely to support internal store operations. We take data handling seriously.

What data is accessed

  • Shop listings — titles, descriptions, pricing, tags, inventory
  • Order and transaction records — order status, fulfilment data
  • Shop-level performance metrics

What we do not do

  • Store customer personal data outside of Etsy systems
  • Share, sell, or distribute any data to third parties
  • Use data for any purpose unrelated to operating the Rekkid store
  • Access data belonging to any shop other than Rekkid

All data accessed through the Etsy API is used strictly for internal operational purposes. Data is retained only as long as needed for analysis and is not sold, shared, or used for advertising or marketing.
